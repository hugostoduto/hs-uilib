# Hs Material Ui

React UI component library

## Installation


Hs Material Ui is available as an [npm](https://www.npmjs.com/package/hs_rng) package.
```bash
npm i hs-uilib
```
## Uso/Exemplos

```javascript
import Button from 'hs-uilib'

function App() {
  return <Button title="Press Me" backgroundColor="#219ebc" handleClick={() => ()} />
}
```
